# PZ Build 42 コレクション管理

Project Zomboid Build 42 のスキルブック / スキルVHS の収集管理ツール。

スマホ・iPad で開いてゲーム中にチェックする用。データは localStorage に保存。

## 使い方

GitHub Pages: `https://<username>.github.io/ZomboidBookKeeper/`

## 機能

- スキルブック 24スキル×5巻 = 120冊の取得数管理
- スキルVHS（経験値付き）の取得管理
- タブで切り替え
- 検索・フィルタ
- +/- ボタンでカウント
- モバイルファースト UI
